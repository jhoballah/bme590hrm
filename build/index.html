avg_hr.new_hr_set(mins, rawbunches, update_time)¶

returns realbunches, groupnum

This function takes in raw instantaneous heart rate based on an averaging window set by the user, as well as minutes specified by user input and returns groupnum and realbunches. Groupnum is the result of taking the user input, converting it to seconds, and then dividing that number by the amount of the instantaneous heart rate window duration (specified in our case as 5 seconds). This is effectively saying the user wants X amount of bunches from the instantaneous HR data. Realbunches is a list that is the instantaneous heart rate array truncated to go to “groupnum” amount of bunched sets.

Parameters:
  • mins (str (from user)) – minutes
  • rawbunches (list) – raw instantaneous heart rate data
  • update_time (int) – windowing (seconds)
Returns:

realbunches - truncated instantaneous heart rate data set
Returns:

groupnum - number of groups to take out of the raw inst HR set, based on user input
Rtype realbunches:
 

list
Rtype groupnum:

int
